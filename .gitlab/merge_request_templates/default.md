## PR 유형
어떤 변경 사항이 있나요?

- [ ] 파일, 폴더 추가
- [ ] 파일 또는 코드, 리소스 제거
- [ ] 단순 변수명 or 파일명 or 경로명을 바꾼 경우
- [ ] 이미지 리소스, prefab 등의 코드와 상관없는 리소스 추가
- [ ] 새로운 기능 추가
- [ ] 버그 수정
- [ ] 문서의 수정
- [ ] (코드의 수정 없이) 스타일(style)만 변경 (들여쓰기 같은 포맷이나 세미콜론을 빼먹은 경우)
- [ ] 주석 추가 및 수정
- [ ] 코드를 리팩토링
- [ ] Test 관련한 코드의 추가, 수정
- [ ] (코드의 수정 없이) 설정을 변경
- [ ] 빌드 부분 혹은 패키지 매니저 수정
- [ ] 브랜치 병합 중 메세지, coflict 해결
